# 自动刷课

## 目标

自动刷国开的视频课程，跳过浪费时间的视频。

## 思路 1

使用 iframe 自动播放

- [ ] 测试 iframe 是否可以播放
  - 不可以，应该是做了相关处理

## 思路 2

打开新标签页播放，同时播放完成后关闭标签页。

主页面

1. 找到所有可点击的 video 标签
2. 逐个点击
3. 声明缓存值 `status` 为 `started`，等待 `status` 的值变为 `compiled`

新标签页

1. 判断是否有 video 标签
2. 如果有则使用 `play` 开始播放
3. 监听播放完成调用关闭函数
4. 监听播放完成
5. 点击下一个
